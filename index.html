Chapter 1: Computer System

1. Computer System Overview

A computer system is an integrated unit of hardware and software. It follows the IPO (Input-Process-Output) cycle.

2. Central Processing Unit (CPU)

The CPU: The main chip that carries out all instructions.
  • ALU (Arithmetic Logic Unit): Performs calculations (add, subtract) and comparisons (Equal to, Less than).
  • CU (Control Unit): Coordinates the flow of data between CPU, memory, and devices.
  • Registers: High-speed internal storage for the CPU to hold data being processed right now.

3. Memory System

A. Primary Memory

  • RAM: Volatile memory that stores data of currently running apps.
  • ROM: Non-volatile; contains permanent start-up instructions (firmware).
  • Cache: Sits between CPU and RAM to store frequently used data for instant access.

B. Secondary Memory

Permanent storage like HDDs (Hard Disks), SSDs, and Pen Drives.

C. Memory Units

1 Bit (0/1) → 8 Bits = 1 Byte → 1024 B = 1 KB → 1024 KB = 1 MB → 1024 MB = 1 GB → 1024 GB = 1 TB.

4. Software & Utilities

System Software

Operating System (OS): Manages hardware and provides a platform for applications (e.g., Windows, macOS, Linux).

Language Processors: Necessary because computers only understand machine code (0s and 1s).

  • Assembler: Converts Assembly language to Machine code.
  • Interpreter: Converts high-level code to machine code line-by-line.
  • Compiler: Converts the entire program into machine code at once.

System Utilities: Software designed to maintain, configure, and optimize the computer system.

  • Antivirus: Protects the system from malicious software.
  • Disk Defragmenter: Reorganizes files on a hard disk for faster access.
  • Compression Tools: Reduces file sizes (e.g., WinZip, 7-Zip).

Application Software

General Purpose: Ready-to-use software like MS Word or Web Browsers.

Customized (Bespoke): Tailor-made for a specific organization (e.g., Railway Reservation System).

5. OS Functions

  • Process Management: Allocating CPU time to different tasks.
  • Memory Management: Tracking which parts of memory are in use.
  • File Management: Keeping track of where data is stored on disks.
  • Device Management: Communicating with hardware via "Device Drivers."

6. Microprocessors & Data Transfer

A microprocessor is a single-chip CPU. Data moves through physical wires called Buses:

  • Data Bus: Carries the actual values being processed.
  • Address Bus: Carries the "location" of where data should go.
  • Control Bus: Carries commands (Read/Write signals).

7. Evolution & Mobile Systems

Computers evolved from Vacuum Tubes (1st Gen) to AI/ULSI (5th Gen). Modern Mobiles use a System on a Chip (SoC) which combines CPU, GPU, and Power Management on one tiny chip.
